# Timeline for iOS
My first project to learn iOS development! This has so far been quite the journey. The idea for this project started about a year or so ago when Google Maps changed their timeline feature. Removing cloud timeline storage ended up costing me my nearly 10 years of timeline data when I forgot to transfer that timeline data to my new phone. 

## Features

### Track visits to your favorite places

### View your visited places on a map

### Automatic visit logging using device location

### One-tap data export to JSON
